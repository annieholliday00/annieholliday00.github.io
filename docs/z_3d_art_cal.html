cal's lightsaber
back
This is a recreation of Cal's lightsaber from Jedi Fallen Order. I wanted to try and create something mostly clean and pristine to see if I could still make it look realistic, but with the warped handle for an added challenge.

A lot of the details on this model are created from normal and height maps, particularly the darker metal inside the top of the lightsaber. I wanted to see how far I could fake the 3d effect without increasing the poly count.
